# 🎓 aT3o - Assistente Acadêmico Inteligente

O **aT3o** é um bot de suporte acadêmico desenvolvido em **Java Spring Boot**. Ele utiliza modelos de Linguagem de Grande Escala (LLMs) para me auxiliar nos meus estudos, mantendo o contexto histórico das conversas para respostas mais precisas e personalizadas.

## 🚀 Tecnologias Utilizadas

* **Java 21** & **Spring Boot 3.x**
* **Spring AI**: Integração com Groq (Llama 3).
* **Google GenAI SDK**: Integração nativa com Google Gemini 1.5 Flash.
* **Spring Data JPA**: Persistência de dados.
* **MySQL**: Banco de dados para histórico de conversas.
* **Lombok**: Produtividade no código.

## 🧠 Funcionalidades

* **Dual-Engine IA**: Suporte a múltiplos provedores (Groq e Google Gemini).
* **Memória de Curto Prazo**: O bot recupera as últimas 10 interações do banco de dados para entender o contexto do diálogo.
* **Personalização Acadêmica**: Configurado via Prompt Engineering para atuar como um tutor de computação e métodos de estudo.
